## About the exam

AZ-900 validates foundational knowledge of cloud concepts and Microsoft Azure. A score of **700 or greater** is required to pass. Skill domains:

| Domain | Weight |
| --- | --- |
| Describe cloud concepts | 25–30% |
| Describe Azure architecture and services | 35–40% |
| Describe Azure management and governance | 30–35% |
